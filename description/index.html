课程体系

CURRICULUM STRUCTURE

课程目标

本课程的学习，共分6个阶段。根据汉字表意性特点，循序渐进系统学习汉字，锻炼用汉字思维、组织文字和读写汉字的能力。整个过程，学员将保持和国内同龄孩子同步的识字进度，累计认识常用汉字2500个，其中2000个左右会写。能对各类少儿读物进行无障碍阅读，能用已学的汉字进行较通顺的常用文体写作。


教材蓝本

我们将部编小学语文课本作为识字蓝本，配套各类中文经典文本（参照《国家教育部+国家图书馆少年儿童馆推荐绘本书目》、《教育部推荐小学生必读书单》）作为辅助教材。


教学内容

课程分阶 学习内容 课时（40分/课）
第一阶段
  1. 认识常用300个汉字，会写其中100个汉字，了解常用汉字部首变体；
  2. 掌握汉字的基本笔画，养成良好的书写习惯；
  3. 认读初级绘本；
  4. 运用已学汉字，看图写话（句子）；
 40课时（6-8个汉字/课时）
第二阶段
  1. 累计认识常用汉字700个左右，累计会写其中300个汉字；
  2. 能较规范地书写汉字；
  3. 认读中级绘本；
  4. 运用已学汉字，看图写话（段落）；
 40课时（8-10个汉字/课时）
第三阶段
  1. 累计认识常用汉字1150个左右，累计会写其中550个汉字；
  2. 能较熟练地书写汉字；
  3. 认读高级绘本；
  4. 运用已学汉字，看图写话（故事）；
 40课时（10-15个汉字/课时）
第四阶段
  1. 累计认识常用汉字1600个左右，累计会写其中800汉字；
  2. 书写规范，有一定的速度；
  3. 认读初级经典文本；
  4. 运用已学汉字，进行100字的话题书面表达；
 40课时（10-15个汉字/课时）
第五阶段
  1. 累计认识常用汉字2100个左右，累计会写其中1300汉字；
  2. 比较规范流畅的书写；
  3. 认读中级经典文本；
  4. 运用已学汉字，进行150字以上的话题书面表达；
 50课时（8-10个汉字/课时）
第六阶段
  1. 累计认识常用汉字2500个，累计会写其中2000汉字；
  2. 比较规范流畅地书写；
  3. 认读高级经典文本；
  4. 运用已学汉字，进行300字的话题书面表达；
 50课时（8-10个汉字/课时）
